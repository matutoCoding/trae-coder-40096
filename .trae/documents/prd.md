## 1. 产品概述

高校科研仪器共享移动App，为高校师生提供科研仪器的在线预约、候补排队、智能计费与账单管理一站式服务。解决仪器资源利用率低、预约冲突、计费不透明等痛点，实现仪器资源的最大化共享与精细化管理。

- 目标用户：高校科研人员、研究生、实验室管理员、设备处管理人员
- 核心价值：提升仪器使用效率，规范预约流程，实现公平透明的分时段计费

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户（师生） | 校园统一认证登录 | 浏览仪器、预约使用、查看账单、候补登记 |
| 仪器管理员 | 管理员指派 | 仪器建档、费率维护、排期管理、资质审核 |
| 系统管理员 | 后台授权 | 全局配置、数据统计、用户管理 |

### 2.2 功能模块

1. **仪器排期模块**：仪器浏览与搜索、时段预约、排期日历、超时自动释放
2. **候补补位模块**：候补排队登记、自动补位通知、候补状态追踪
3. **时段计费模块**：时段费率表维护、跨档时长拆分、实时费用预估
4. **账单生成模块**：使用记录汇总、账单自动生成、账单明细查询

## 3. 核心流程

### 3.1 预约与超时释放流程

用户浏览仪器 → 查看排期 → 选择时段 → 资质校验 → 确认预约 → 预约成功（锁定时段）→ 到达签到开始使用 → 使用结束结算 → 生成账单

若预约后超时未签到：超时检测（预约开始后15分钟）→ 自动释放时段 → 触发候补补位

```mermaid
flowchart TD
    "用户浏览仪器" --> "查看排期日历"
    "查看排期日历" --> "选择可用时段"
    "选择可用时段" --> "资质校验"
    "资质校验" -->|"通过"| "确认预约"
    "资质校验" -->|"未通过"| "提示资质不足"
    "确认预约" --> "锁定时段"
    "锁定时段" --> "等待签到"
    "等待签到" -->|"15分钟内签到"| "开始使用"
    "等待签到" -->|"超时未签到"| "自动释放时段"
    "自动释放时段" --> "通知候补队列首位"
    "通知候补队列首位" -->|"确认补位"| "锁定时段"
    "通知候补队列首位" -->|"放弃/超时"| "通知下一位候补"
    "开始使用" --> "使用结束"
    "使用结束" --> "跨档计费拆分"
    "跨档计费拆分" --> "生成账单"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝 #1A365D（科技感、专业感），辅色：琥珀橙 #ED8936（活力、提醒）
- 按钮风格：圆角胶囊按钮，主按钮填充色，次按钮描边
- 字体：标题使用思源黑体 Bold，正文使用系统默认字体
- 布局风格：移动端优先，卡片式布局，底部Tab导航
- 图标风格：线性图标，2px描边，统一风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 仪器推荐 | 渐变背景头部、横向滑动卡片、圆角搜索框、标签筛选 |
| 仪器详情页 | 排期日历 | 时间轴式日历、色块标注可用/已约/候补、底部预约按钮 |
| 预约确认页 | 时段选择 | 时间滑块选择器、实时费用预览卡、资质状态标签 |
| 候补登记页 | 队列展示 | 垂直排队卡片、序号徽章、预计等待时间倒计时 |
| 补位通知页 | 通知卡片 | 倒计时确认按钮、仪器缩略图、时段信息 |
| 费率管理页 | 费率表 | 时段色彩条、编辑弹窗、拆分预览动画 |
| 账单列表页 | 账单卡片 | 月度汇总卡、状态标签、展开式明细 |
| 账单详情页 | 费用明细 | 分段计时条、各段费率标注、合计金额突出 |

### 4.3 响应式设计

- 移动端优先设计（375px基准宽度）
- 触控优化：按钮最小44px点击区域，滑动操作友好
- 适配屏幕：375px~428px手机屏幕为主，兼顾平板横屏
